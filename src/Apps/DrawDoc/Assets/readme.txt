This folder stores SVG images locally to be used as custom mouse cursors
The icons are not originals, but altered versions with the same name.
The alterations have been made are the following:
-Altered Background opacity to 0
-Rotated by 180 degrees
-Added stroke to Black and stroke width to 20
-Resized image over 128px to comply with firefox hardcoded limit
Sources for the originals:
Themify Icons: https://github.com/lykmapipo/themify-icons
Game Icons: https://game-icons.net/
